# 🚀 Quick Deploy to GitHub Pages

## Fastest Way (5 minutes)

### Step 1: Copy the file
```bash
cd C:\Users\mayso\artsoul-marketplace
cp index.html C:\Users\mayso\maysonkiller.github.io\artsoul.html
```

### Step 2: Commit and push
```bash
cd C:\Users\mayso\maysonkiller.github.io
git add artsoul.html
git commit -m "Add ArtSoul Marketplace"
git push origin main
```

### Step 3: Access your site
Visit: **https://maysonkiller.github.io/artsoul.html**
