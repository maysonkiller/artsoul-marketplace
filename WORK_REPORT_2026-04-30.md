## 🎯 НОВАЯ ЛОГИКА МАРКЕТПЛЕЙСА

### Как работает аукцион:

**1. Создание аукциона (3 дня):**
```
Художник → Создает аукцион → Статус: AUCTION
```

**2. Размещение ставок:**
```
Участник → Делает ставку 1 ETH → Платит депозит 0.1 ETH
→ Предыдущий участник получает депозит обратно
→ Новый участник становится highest bidder
```

**3. Завершение аукциона:**
```
Аукцион завершается → Floor price = 1 ETH (highest bid)
→ Победитель получает 24 часа на покупку
→ Статус: AUCTION_ENDED
→ Все депозиты возвращаются (кроме победителя)
```

**4. Покупка победителем (24 часа):**
```
Победитель → Платит 0.9 ETH (1 ETH - 0.1 ETH депозит)
→ NFT минтится и передается ему
→ Статус: SOLD
→ Роли: Creator, Auction Winner (он же), Current Owner (он же)
```

**5. Если победитель НЕ покупает:**
```
24 часа истекли → Статус: FOR_SALE
→ Художник выставляет на прямую продажу (цена >= 1 ETH)
→ Любой может купить напрямую
```

**6. Прямая покупка:**
```
Покупатель → Покупает за 1.5 ETH
→ NFT минтится и передается ему
→ Статус: SOLD
→ Роли: Creator (художник), Auction Winner (первый победитель), Current Owner (новый покупатель)
```

### Три роли:

1. **Creator** (создатель):
   - Всегда один и тот же
   - Получает оплату при продаже
   - Может выставлять на продажу после аукциона

2. **Auction Winner** (победитель аукциона):
   - Почетное звание
   - Устанавливается один раз
   - НЕ меняется при перепродажах
   - Показывается на странице artwork

3. **Current Owner** (текущий владелец):
   - Владелец NFT
   - Меняется при каждой продаже
   - Может перепродавать NFT
